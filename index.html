BulletSharp

.NET wrapper for the Bullet physics library

Download this project as a .zip file Download this project as a tar.gz file

BulletSharp is a complete .NET wrapper for the Bullet physics library written in C++/CLI.

It has bindings to Mogre, MonoGame, OpenTK, SharpDX and SlimDX. The stand-alone Generic package includes its own math classes.

Binaries:

  • bulletsharp-2.83.7.1.zip
  • bulletsharp-x64-2.83.7.1.zip
  • bulletsharp-demos-2.83.7.1.zip

Try the new experimental cross-platform version for Linux and Windows:

  • bulletsharp-pinvoke-0.7.zip
  • bulletsharp-pinvoke-demos-0.7.zip
  • NuGet

See also: wiki pages.

Note: BulletSharp will not work on the Xbox 360 or any other fully managed environment, because it uses the native Bullet library. An alternative may be Bullet-XNA.
