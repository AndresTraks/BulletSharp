BulletSharp

.NET wrapper for the Bullet physics library

Download this project as a .zip file Download this project as a tar.gz file

BulletSharp is a complete .NET wrapper for the Bullet physics library written in C++/CLI.

It has bindings to Axiom, Mogre, OpenTK, SharpDX, SlimDX, MonoGame, XNA (to be deprecated) and Windows API Code Pack. The stand-alone Generic package includes its own math classes.

Binaries:

  • All Downloads

Try the new experimental cross-platform version for Linux and Windows:

  • bulletsharp-pinvoke-0.5.zip
  • bulletsharp-pinvoke-demos-0.5.zip

Note: BulletSharp will not work on the Xbox 360 or any other fully managed environment, because it wraps the Bullet library, which is unmanaged. An alternative may be Bullet-XNA.
